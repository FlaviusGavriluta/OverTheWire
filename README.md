# OverTheWire - Bandit

## Level Goal
The goal of this level is to log into the game using SSH. 

- **Host:** `bandit.labs.overthewire.org`  
- **Port:** `2220`  
- **Username:** `bandit0`  
- **Password:** `bandit0`  

Once logged in, go to the [Level 1](https://overthewire.org/wargames/bandit/bandit1.html) page to find out how to beat Level 1.

---

## Commands You May Need
- `ssh` - to connect to the server
